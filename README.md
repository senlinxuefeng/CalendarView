# CalendarView
这是一款可以选择开始日期和结束日期的日历哦。
在做项目时，遇到了日历选择器的问题，开始也是在网上找了很多相似的demo，始终没有找到一款合适自己项目的，为了满足项目的要求，
在别人的一个demo基础进行了大量修改，终于达到了项目的要求。
